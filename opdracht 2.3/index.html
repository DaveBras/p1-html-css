Waarom noem je de eerste HTML-pagina; index.html?

De startpagina van een website heet meestal index.html omdat dit de standaardnaam is voor de startpagina van een website op een server.

Met welke versie HTML werken we nu?

De meest recente versie van HTML en dus de versie die we gebruiken is HTML5

Met welke CSS versie werken we nu

Wij gebruiken versie 3 van CSS

Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Dit komt doordat elke browser zijn eigen manier heeft om HTML en CSS te interpreteren en weer te geven. Hierdoor kunnen kleine verschillen ontstaan in de lay-out, kleuren en lettertypen van een website.
